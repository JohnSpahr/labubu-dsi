## how to install
watch <a href="https://youtu.be/sWGuQGQDCD0?si=9VGVYiO-_62IwXMj">this video</a> or follow the steps below:
1. install cfw on your DSI (/XL) using <a href="https://dsi.cfw.guide">this guide</a>.
2. Go to SD card/_nds/TWiLightMenu/extras and paste splashtop and splashbottom GIFs. This adds a custom splash screen when you boot the system
3. Copy the theme folder to SD card/_nds/TWiLightMenu/3dsmenu/themes
4. Next, boot your DSI and press SELECT in TWiLightMenu
5. It should take you to the classic DS home screen. Go ahead and click the button with the DS at the bottom middle of the lower screen. This will take you to the homebrew settings page
6. Next, set "user interface" under GUI settings to "Nintendo 3DS"
7. Then, click "custom theme" and select the Labubu theme
8. Finally, make sure "DSi/3DS music" is set to custom

And like that, you should be set!

"Labubu-dsi" is based on <a href="https://skins.ds-homebrew.com/nintendo-3ds/blue">this theme</a>!
